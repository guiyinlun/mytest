**七巧板小猫**

七巧板是一种古老的中国传统智力玩具，其历史至少可以追溯到公元前一世纪，到了明代基本定型。明、清两代在中国民间广泛流传，清陆以湉《冷庐杂识》卷一中写道：近又有七巧图，其式五，其数七，其变化之式多至千余。体物肖形，随手变幻，盖游戏之具，足以排闷破寂，故世俗皆喜为之。” 在 18 世纪，七巧板流传到了国外。李约瑟说它是东方最古老的消遣品之一，至今英国剑桥大学的图书馆里还珍藏着一部《七巧新谱》。
